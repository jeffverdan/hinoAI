Cifra block — fixed-width chords aligned above lyric lines. Align chords with spaces in the `chords` string.

```jsx
<ChordSheet section="Refrão" lines={[
  { chords: "G              C       D", text: "Cantai ao Senhor um cântico" },
  { chords: "Em           C        G", text: "novo, pois fez maravilhas" },
]} />
```

Monospace alignment is intentional — keep chord/lyric strings space-padded.
