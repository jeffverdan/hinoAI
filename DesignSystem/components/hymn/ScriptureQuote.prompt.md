Hino.AI's signature device — a Bible verse set in italic Spectral with the reference after an em-dash. Use sparingly, as a held note.

```jsx
<ScriptureQuote verse="Cantai ao Senhor um cântico novo." reference="Salmos 96:1" size="lg" align="center" halo />
```

Sizes `sm | md | lg | xl`. Add `halo` for hero/reveal moments. Quotes are added automatically — don't include them in `verse`.
