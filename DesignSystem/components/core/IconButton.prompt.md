Circular button holding a single icon — toolbars, the player transport, dismiss buttons. Always pass `label` for accessibility.

```jsx
<IconButton icon="play" variant="solid" size="lg" label="Tocar" />
<IconButton icon="heart" label="Favoritar" />
<IconButton icon="x" variant="outline" label="Fechar" />
```

Variants: `ghost` (default), `outline`, `solid` (gold). Sizes `sm 36 · md 44 · lg 56`.
