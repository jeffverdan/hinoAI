Primary action control — pill-shaped, warm gold by default. Use `primary` for the single main action per view; `secondary`/`ghost` for supporting actions.

```jsx
<Button iconLeft="sparkles">Criar meu hino</Button>
<Button variant="secondary">Ouvir um exemplo</Button>
<Button variant="ghost" size="sm" iconRight="chevronRight">Pular</Button>
```

Variants: `primary` (gold), `secondary` (outline), `ghost`, `gold-soft` (tinted), `danger`. Sizes `sm | md | lg`; `full` to stretch. Use one primary per screen.
