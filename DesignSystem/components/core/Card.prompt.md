Surface container — white on ivory with a warm soft shadow, gently rounded (18px). Works on both themes.

```jsx
<Card>…</Card>
<Card interactive onClick={...}>Hino salvo</Card>
<Card selected pad="lg">Estilo escolhido</Card>
```

Props: `pad` none|md|lg, `interactive` (hover lift), `selected` (gold glow ring), `elevation` none|sm|lg.
